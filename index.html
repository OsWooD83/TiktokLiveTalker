Comment ça marche

Configurez en quelques clics et laissez la synthèse vocale gérer vos messages TikTok.

1

Connectez votre live TikTok

Entrez le nom de votre live pour récupérer le chat en temps réel.

2

Lancez la lecture

Cliquez sur « Start » pour activer la lecture vocale des nouveaux messages.

3

Laissez parler

Les messages s'affichent et sont lus automatiquement. Vous restez concentré sur l'animation.

Installation

Prêt à lancer TiktokLiveTalker

ZIP → extraction → double-clic sur run.bat : les dépendances s'installent et l'app démarre seule.

  1. Téléchargez l'archive ZIP puis décompressez-la dans un dossier de votre choix.
  2. Ouvrez le dossier extrait et double-cliquez sur TiktokLiveTalker-1.0.0.
  3. La fenêtre d'installation télécharge les dépendances nécessaires, puis lance automatiquement l'application.
  4. Une fois ouverte, saisissez URL de votre live TikTok, choisissez la voix et cliquez sur « Start ».
  5. l'application sera visible sur votre bureau
